Visualización de Redes con Diferentes Tecnologías

Este proyecto explora diferentes formas de crear una red de nodos interactiva utilizando varias tecnologías de visualización web. Cada implementación mantiene el mismo concepto básico pero aprovecha las características únicas de cada tecnología.

p5.js 2D

Implementación básica en 2D utilizando p5.js, perfecta para visualizaciones simples y eficientes.

p5.js 3D

Versión 3D utilizando WEBGL en p5.js, añadiendo profundidad a la visualización.

WebGL Puro

Implementación utilizando WebGL directamente, ofreciendo mayor control y rendimiento.

Three.js

Versión avanzada utilizando Three.js, con iluminación, sombras y efectos 3D más sofisticados.

Guía de Solución de Problemas

  • Asegúrate de que todos los scripts se cargan correctamente (revisa la consola del navegador).
  • Para WebGL y Three.js, verifica que tu navegador y tarjeta gráfica soporten WebGL.
  • Si las páginas aparecen en blanco, intenta añadir un console.log al inicio de tus scripts para verificar que se están ejecutando.
  • Asegúrate de que los elementos canvas tienen dimensiones explícitas o están configurados para ocupar el espacio disponible.
